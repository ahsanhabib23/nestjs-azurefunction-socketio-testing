## Installation

```bash
$ git clone https://github.com/ahsanhabib23/nestjs-azurefunction-socketio-testing.git
$ cd nestjs-azurefunction-socketio-testing
$ npm install
$ npm run start:dev (to run basic nestjs application locally)
$ npm run start:azure (to run basic nestjs application with azure function integrated locally)
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
